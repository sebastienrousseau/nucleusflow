Expand description

Core trait for implementing content processors.

This trait defines the interface for any type that can process content from one form to another. It’s designed to be flexible and composable through generic type parameters.

§Type Parameters

  • Input: The type of content being processed
  • Output: The type of content produced
  • Context: Additional context or configuration for processing
